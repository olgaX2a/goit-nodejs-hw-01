# goit-node-hw-01

Commands used for test: 
- [x] **get all contacts** 📦️  
```
node index -a list
```
- [x] **get user by id** ✅  
```
node index -a get -i 7
```
- [x] **remove user by id** ❌  
```
node index -a remove -i 7
```
- [x] **add user** ➕  
```
node index -a add -n Geralt -e geralt@mail.com -p 012-345-67-89
```
